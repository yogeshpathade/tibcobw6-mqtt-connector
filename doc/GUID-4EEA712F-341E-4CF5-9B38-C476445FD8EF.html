Deploying Applications

After deploying applications, you can manage BusinessWorks applications by using TIBCO Enterprise Administrator.

Prerequisites

The following tasks are required before deploying applications:
  • Creating a Project.
  • Generating an EAR File.
A typical workflow of deployment includes:
  1. Upload an EAR file.
  2. Deploy an application.
  3. Configure an application.
  4. Start an application.

You can deploy an application EAR file by using the command-line mode with the bwadmin utility. See TIBCO ActiveMatrix BusinessWorks Administration for more details about how to deploy an application.
